Dashboard Management
How do I access my dashboard?
Log in to your InextLabs account and you'll be automatically directed to your main dashboard. You can also access it anytime by clicking "Dashboard" in the navigation menu.

How do I reset my dashboard metrics?
To reset your dashboard metrics and start fresh:
1. Navigate to your Dashboard
2. Click on the "Settings" icon (gear icon) in the top right corner of the metrics panel
3. Select "Reset Metrics" from the dropdown menu
4. Confirm the reset action in the popup dialog
5. Your metrics will be reset to zero and start tracking from the current moment
Note: This action cannot be undone. Historical data will be archived but not visible in the main dashboard view. You can access archived metrics in Dashboard Settings → Analytics History.

How do I customize my dashboard?
You can customize your dashboard layout by:
1. Go to Dashboard → Customize Layout
2. Drag and drop widgets to rearrange them
3. Click the "+" button to add new widgets
4. Click the "x" on any widget to remove it
5. Click "Save Layout" to preserve your changes
Available widgets include: API Usage, Model Performance, Cost Analytics, Request Timeline, Error Logs, and Custom Metrics.

How do I export dashboard data?
To export your dashboard data:
1. Navigate to Dashboard → Analytics
2. Select the date range you want to export
3. Click "Export Data"
4. Choose format (CSV, JSON, or PDF)
5. Click "Download"
The export will include all metrics, usage statistics, and performance data for the selected period.

Can I share my dashboard with team members?
Yes. Professional and Enterprise plans support team collaboration:
1. Go to Dashboard → Team Settings
2. Click "Invite Team Member"
3. Enter their email address
4. Set their permission level (View Only, Editor, or Admin)
5. Click "Send Invitation"
Team members will receive an email invitation to access the shared dashboard.

How do I set up dashboard alerts?
To configure alerts for important metrics:
1. Navigate to Dashboard → Alerts
2. Click "Create New Alert"
3. Select the metric to monitor (e.g., API usage, error rate, cost threshold)
4. Set the trigger condition (e.g., "when usage exceeds 80%")
5. Choose notification method (email, SMS, or webhook)
6. Click "Save Alert"
You'll receive notifications when alert conditions are met.

Common Issues and Troubleshooting
"Authentication Failed" Error
Verify your API key is correct
Check if your key has expired
Ensure you're using the correct authentication header format
Confirm your account is active and in good standing

"Rate Limit Exceeded" Error
Check your current usage in the dashboard
Upgrade your plan for higher limits
Implement request throttling in your application
Contact support for temporary limit increases

"Invalid Request Format" Error
Review API documentation for correct request structure
Verify JSON formatting
Check required parameters are included
Ensure content-type headers are set correctly

Slow Response Times
Check your internet connection
Verify regional endpoint usage (use closest region)
Review your request complexity
Consider caching frequently used results
Contact support if issues persist

Model Not Returning Expected Results
Review input data quality and formatting
Check model parameters and settings
Verify you're using the correct model for your use case
Review examples in documentation
Consider fine-tuning for better results

Dashboard Not Loading or Showing Errors
Clear your browser cache and cookies
Try accessing the dashboard in an incognito/private window
Check if you're using a supported browser (Chrome, Firefox, Safari, Edge)
Disable browser extensions that might interfere
Check status.inextlabs.com for any ongoing service issues
If problems persist, contact support with your browser and operating system details

Dashboard Metrics Not Updating
Ensure your API key is active and correctly configured
Check if you've reached your plan's rate limits
Verify your application is successfully making API calls
Refresh the dashboard page (metrics update every 5 minutes)
Check Dashboard Settings → Refresh Interval and adjust if needed
